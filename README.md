##  多因素皮尔逊分析软件

### 程序说明

可运行程序位于dist目录下，dist->Pearson.exe

### 使用说明

+ 读取文件为excel

+ 表格形式如下：

  | 井名    | *因素一$m^3 | ...  |
  | ------- | ----------- | ---- |
  | *井名一 | *数字值     |      |
  | *井名二 | *数字值     |      |
  | ...     |             |      |

  

+ 若想在因素中使用带上下标或公式等内容，请在内容前加上'$'符号（如上表因素一），使用格式为LaTex公式排版格式
+ 值可为0或空值
