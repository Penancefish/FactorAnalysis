##  多因素分析软件

### 程序说明

可运行程序位于压缩包中，下载即可使用，目前仅支持windows-64位

### 使用说明

+ 读取文件为excel

+ 表格形式如下：

  | 井名    | *因素一$m^3 | ...  |
  | ------- | ----------- | ---- |
  | *井名一 | *数字值     |      |
  | *井名二 | *数字值     |      |
  | ...     |             |      |

  

+ 若想在**皮尔逊因素分析成图**中显示带上下标或公式等内容，请在内容前加上'$'符号（如上表因素一），使用格式为LaTex公式排版格式

+ 除井名一栏外，值可为0或空值，但**不可为其它字符**

+ 对于**灰色关联分析**功能，会将**最后一列**因素视为分析对象，使用时请注意
